## Project structure

```
hndwok-fe-web
├── public(all public assets e.g images etc.)
├── src (other folders)
    ├── components
    ├── layouts
    ├── hooks
    ├── pages
    ├── services
    ├── store
    ├── styles
    └── utilities
└── ...(consits of config files, env file, package.json etc.)

```

`src` folder is for the web application components, containing pages, styles, services etc..
